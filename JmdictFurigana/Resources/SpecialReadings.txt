; This is a text file containing words that have a special reading (熟字訓).
; These readings may attribute a furigana string to several characters indissociably.
; A good example of this is:
; 田舎|いなか
; You cannot cut it in い.なか or いな.か - the whole expression should have the furigana いなか.
; Now the reason we do not put these in the OverrideFurigana file is that they sometimes come up
; as part of other words. Example: 田舎家.

; The format here is:
; <Kanji reading>|<Kana reading>(|<Solution string>)
; Specifying the solution string should be done when the special reading is split over multiple
; characters (e.g. 其処 (そこ) because the reading can be cut in 其=そ, 処=こ). Otherwise, it
; is assumed that the kana reading covers the whole kanji string.
; You can define multiple entries to allow multiple readings for a same kanji reading.
